# CMSTemplate
一个由react + react-redux + antd + node写的后台管理系统

- 技术站： 
    - 前端：react + react-redux + antd + axios
    - 服务： node


### 项目启动

- 前端页面
1. 安装依赖包： 
```
    npm install
```
2. 启动 ： 
```
    npm start
```
3. 服务器监听3000端口，直接访问 http://localhost:3000 
- 后台服务
1. 安装依赖包： 
```
    npm install
```
2. 启动 ： 
```
    node server.js
```
